# 🔄 User Flow

---

1. Create an account or sign in.
2. Create a new shelf or select an existing one.
3. Add media items by entering details such as title, creator, media type, rating, and status.
4. Track your progress and update entries as you read, watch, or play.
5. View insights through the analytics dashboard.
6. Optionally make shelves public and discover collections shared by other users.
7. Interact with the community by exploring and rating public shelves.

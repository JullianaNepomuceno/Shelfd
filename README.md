# Shelfd

Shelfd is a media tracking and discovery platform where users can organize movies, books, comics, TV shows, and games into personalized shelves. Track progress, manage backlogs, rate media, and discover highly rated public shelves from the community.

---

## Tech Stack

- **Backend:** Java, Spring Boot, Spring Security, Spring Data JPA
- **Database:** PostgreSQL
- **Frontend:** React, TypeScript, HTML, CSS
- **Build Tools:** Maven
- **Other:** Lombok

---

## Features

- User login and registration
- Create and manage shelves
- Add movies, books, comics, TV shows, and games
- Upload or add cover images through links
- Track media status:
  - Finished
  - Unfinished
  - In-Progress
- Edit or delete media entries
- Personal ratings and comments
- Search and filter media entries
- Public and private shelves
- Community shelf rating system
- Top rated shelves leaderboard
- Personalized dashboard statistics
- Dark mode support

---

## Competitive Features

Shelfd combines personal media tracking with community-driven discovery features.

### Personalized Analytics Dashboard
Users can visualize their media habits through:
- Total media tracked
- Completed vs unfinished media
- Average ratings
- Media category breakdown
- Progress tracking per media type

### Public Shelf Sharing
Users can make shelves public and share them with others.

### Community Shelf Ratings
Users can rate public shelves created by other users, encouraging interaction and discovery within the platform.

### Top Rated Shelves
A leaderboard-style page showcases the highest-rated public shelves in the community, helping users discover curated recommendations and popular collections.

### Media Discovery Experience
Search and filter systems help users quickly browse media entries and explore community-created shelves.

The project focuses on combining clean UI/UX, full-stack development, data visualization, and interactive social features into a polished media tracking platform.

---

## User Flow

1. Create or log into your account.
2. Access your personal dashboard.
3. Create a new shelf or select an existing shelf.
4. Add media entries by providing:
   - Media title
   - Media type
   - Creator/author
   - Cover image
   - Media link (optional)
   - Personal rating
   - Comment/review
   - Status (Finished, Unfinished, In-Progress)
5. Edit or delete media entries anytime.
6. Optionally make shelves public for community viewing.
7. Browse and rate public shelves created by other users.
8. Discover trending and top-rated shelves through the leaderboard page.

## Goal

Shelfd aims to provide a clean and interactive platform for managing personal media collections while encouraging community-driven discovery and recommendations.

# Shelfd 📚🎬🎮

### Never lose track of what you're reading, watching, or playing again.

**Shelfd** is a media tracking platform that helps users organize books, comics, movies, TV shows, and games into personalized digital shelves. Whether you're managing a growing backlog, tracking your progress, or discovering new favorites, Shelfd keeps your entire media journey in one place.

---

## 🛠️ Tech Stack

- **Backend:** Java, Spring Boot, Spring Security, Spring Data JPA
- **Database:** PostgreSQL
- **Frontend:** React, TypeScript, HTML, CSS
- **Build Tools:** Maven
- **Other:** Lombok


---

## ✨ Core Features

### User Authentication

* Secure user registration and login
* Personalized user profiles

### Shelf Management

* Create and manage multiple shelves
* Organize media based on your preferences

### Media Tracking

* Add books, comics, movies, TV shows, and games
* Store media details, ratings, and progress

### Progress Status

Track your media with customizable statuses:

* Completed
* In Progress
* Planned / Backlog

### Media Management

* Edit existing entries
* Remove media from shelves
* Rate and review media items

### Shelf Sharing

* Share shelves publicly or keep them private
* Showcase your collections and recommendations

---

## 🚀 Competitive Features

Shelfd goes beyond traditional media trackers by combining personal organization with social discovery.

### 📊 Personalized Analytics Dashboard

Gain insights into your media habits through visual statistics, including:

* Total media tracked
* Completion rates
* Average ratings
* Media category distribution
* Reading, watching, and gaming trends

### 🌎 Public Shelf Sharing

Users can make their shelves publicly accessible, allowing others to explore curated collections and recommendations.

### ⭐ Community Shelf Ratings

Browse public shelves and rate collections created by other users, encouraging community interaction and content discovery.

### 🔍 Discover New Media

Explore public shelves to find books, shows, movies, comics, and games recommended by other users with similar interests.

---

## 🎯 Project Goal

Shelfd aims to provide a clean, intuitive, and engaging platform that combines media tracking, analytics, and community-driven discovery. By bringing personal organization and social interaction together, Shelfd helps users build their own digital shelf while connecting with fellow media enthusiasts.

---

## 🔄 User Flow

1. Create an account or sign in.
2. Create a new shelf or select an existing one.
3. Add media items by entering details such as title, creator, media type, rating, and status.
4. Track your progress and update entries as you read, watch, or play.
5. View insights through the analytics dashboard.
6. Optionally make shelves public and discover collections shared by other users.
7. Interact with the community by exploring and rating public shelves.
